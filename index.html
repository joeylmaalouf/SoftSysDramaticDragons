Project 1: Shell

Joey Maalouf, Apurva Raman, Sean Carter

Abstract

For this project, we aimed to create a functional shell, that would be able to access and use built in linux commands. Our implimentation allows for many of the basic features that a user might expect; these are features such as an interactive terminal mode, a batch file mode, configuration files, and aliases. Basic features of the parsing include multiline quotes and comments, commands separated by semicolons, and more. It also, of course, features a customizable prompt.

Background

Most of our reasearch and understanding comes from the bash shell that is used by default in most linux distributions. While trying to identify essential features, we refered to the bash reference manual provided by gnu.org. We were also inspired by a number of assignments used at other colleges, such as this one from Perdue.
